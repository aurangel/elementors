Дизайнер отчетов

 
Программа обладает очень богатым функционалом для создания любых видов отчетов, а также широким спектром операций над ними. Вы можете создавать отчеты любой сложности. У многих кнопок имеются всплывающие подсказки, поэтому о них здесь упоминаний не будет. О том, как взаимодействовать с редактором читайте здесь.
 

Стандартные операции

Стандартные операции.
  • Отчет
  • Создать пустой отчет.
  • Открыть отчет с расширением *.repx.
  • Сохранить текущий отчет.
  • Сохранить все измененные отчеты.
  • Правка
  • Вырезать / копировать / вставить элементы отчета.

Настройки элементов

  • Шрифт
  • Настройка шрифтов, расположение по сторонам, цвет и так далее.
  • Выравнивание
  • Выравнивание элементов по ширине.
  • По левому краю.
  • По центру.
  • По правому краю.
  • Выравнивание верхушки.
  • Средней части.
  • Низней части.

Элменты в описании не нуждаются. Чтобы активировать их, перетяните элемент на отчет.
