Практическое руководство

Создание отчета

 
Создание простейшего отчета, в который будет также включена выборка произвольных данных из реляционных таблиц БД.
 

Запуск

В главном окне программы выберите "Дизайнер отчетов".
Наверх

Создание отчета с запросом

Нажмите сочетание клавиш Ctrl+W или же во всплывающем списке элемента "Новый отчет" выберите "Новый отчет с помощью мастера". Выберите "Data-bound Report".
Наверх

Строка подключения

Для запроса необходимо сперва настроить строку подключения.
 
  1. Тип вашей базы базируется на СУБД, из которой вы хотите загрузить данные в отчет и может быть одним из следующих:
  • Microsoft SQL Server
  • Microsoft Access 2003
  • Microsoft Access 2007
  • Oracle
  • XML
  • SAP Sybase ASE
  • IBM DB2
  • Firebird
  • MySQL
  • PostgreSQL
  • SQLite
 
2. Путь к базе.
Наверх

Создание запроса

Создание запросов тоже оформляется дизайнером и не составляет труда. В диалоговом окне мастера нажмите "Run Query Designer". Вы увидите следующее:
  1. Все таблицы, которые были найдены в вашей базе данных.
  2. Выбранные строки. Просто перетяните их имена в этот контейнер.
  3. На основании тех строк, что вы выберете, будет составляться запрос на структурированном языке запросов SQL. Посмотреть его можно нажав кнопку "Просмотр".
Наверх

Выбранные поля

На этом окне необходимо активировать поля, которые вы выбрали в предыдущем окне.
Наверх

Поля БД

Перенесите необходимые поля БД на страницу отчета. Это необходимо для того, чтобы немедленно перенести запрошенные строки в отчет.
Наверх

Шаблон отчета

Стиль отчета задается несколькими шаблонами. Среди них можно выделить:
  • Полужирный
  • Простой
  • Корпоративный
  • Компактный
  • Официальный
Наверх

Готовность

На последнем диалоговом окне мастера введите имя будущего отчета. Перейдите на вкладку "Предварительный просмотр" и выберите тип бумаги "Размер -> А4".
 
На этом этапе можно считать отчет готовым. Вы можете взаимодействовать с прочими элементами как угодно.
